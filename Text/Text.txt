Приложение предназначено для управления списком пользователей с использованием данных из MongoDB и взаимодействием с веб-сервером через API.

Основные компоненты:
MainViewModel (основная ViewModel):

Связывает данные (модель пользователя) с интерфейсом (View).
Содержит коллекцию пользователей ObservableCollection<User>, которая автоматически обновляет интерфейс при добавлении или изменении данных.
Свойства для ввода имени, email и возраста.
Команда AddUser позволяет добавлять новых пользователей в коллекцию и отправлять их на сервер через метод NetController.WriteUser(user).
NetController:

Содержит методы для взаимодействия с сервером через HTTP-запросы:
GetAllUsers() - получает список всех пользователей с сервера.
WriteUser(User user) - отправляет нового пользователя на сервер.
User (модель пользователя):

Модель данных пользователя, содержащая свойства: Id, Name, Email, и Age.
Интерфейс (XAML):

Содержит ListView, отображающий список пользователей с именем, email и возрастом.
Три текстовых поля для ввода имени, email и возраста.
Кнопка Add, которая добавляет пользователя в список и отправляет данные на сервер.
Как это работает:
Загрузка пользователей:

В конструкторе MainViewModel загружается список пользователей с сервера с помощью NetController.GetAllUsers().
Ответ в формате JSON преобразуется в массив BSON-объектов и добавляется в коллекцию Users.
Добавление пользователей:

Пользователь вводит данные (имя, email и возраст) в текстовые поля.
При нажатии на кнопку Add, срабатывает команда AddUser, которая:
Создает нового пользователя.
Добавляет его в локальный список Users.
Отправляет данные на сервер для сохранения через NetController.WriteUser(user).
Интерфейс:

ListView динамически отображает список пользователей, который обновляется при добавлении новых данных.
Пользователи могут добавляться через интерфейс WPF, и данные будут синхронизированы с сервером.
